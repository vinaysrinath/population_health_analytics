# CS 6440 Team 20

## Webapp Project Details

### Technology Stack

* Spring MVC 4.2.2
* Maven 3.0.4
* Jquery 2.1.4
* Highcharts 4.1.9
* Tomcat 8

### Deployment

Compile and genrate WAR file from the project and deploy into webapps folder in Tomcat

### Usage

The User Interface is accessible from 

http:&lt;host&gt;:&lt;port&gt;/PopHealth
